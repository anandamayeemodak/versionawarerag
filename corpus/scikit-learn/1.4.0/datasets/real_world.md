Real world datasets
===================

scikit-learn provides tools to load larger datasets, downloading them if
necessary.

They can be loaded using the following functions:

   fetch_olivetti_faces
   fetch_20newsgroups
   fetch_20newsgroups_vectorized
   fetch_lfw_people
   fetch_lfw_pairs
   fetch_covtype
   fetch_rcv1
   fetch_kddcup99
   fetch_california_housing
